Welcome, GM!

Add Character

Create a new character using the POST /gm/add-character/ endpoint.

Add Ability

Assign abilities to characters with the POST /gm/add-ability/ endpoint.

Add Expertise

Provide expertise to characters via the POST /gm/add-expertise/ endpoint.

Add Job Skill

Enhance characters with job skills using POST /gm/add-job-skill/.

Add To Inventory

This leads to a form where you can create both Items and Weapons to be added to their resective database. The form will adjust based on the inventory value being added.

Add Species Passive

Assign species passives via the POST /gm/add-species-passive/ endpoint.

View Characters

See all created characters using GET /gm/view-characters/.

View Libraries

Explore available libraries using GET /gm/view-libraries/.

View Backpacks

Check player backpacks with GET /gm/view-backpacks/.

View Item Inventory

Access the item inventory with GET /gm/view-items/.

View Abilities

Manage abilities using GET /gm/view-abilities/.

Bulk Remove Items

Delete multiple items at once with DELETE /gm/remove-items/.
